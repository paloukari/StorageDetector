Welcome

Welcome to my HTML Guide -- I hope you find it useful :)

This easy guide for beginners covers several topics, with short and basic descriptions of the HTML tags you are likely to need when learning how to make your own website.

A good place to start would be the essential tags section, followed by the basics, which should give you enough information to write your first page.

What is html? What are tags? - Perhaps check out the what is html section first.

Don't miss the HTML cheat sheet for a quick reference list of HTML tags and their attributes all on one page. There is also the CSS cheat sheet for style sheets.

Learn by Example

Often, the best way to learn is by example, and so you will find many samples such as this one, which shows the basic html structure.

<html>
 <head>
  <title>My Title</title>
 </head>
 <body>
  <p>Hello World</p>
 </body>
</html>

Try out the examples using the live demo, experiment by entering your own tags to see how they work.

Stylesheets (CSS)

Cascading Style Sheets (CSS) are the preferred way for setting the look and feel of a website. The stylesheet defines the colour, size and position of elements, while the HTML defines the content and how it is organised.
